# msrproducts Plugin Matrix

## Required
- WooCommerce: product catalog data model and product templates.
- WP Mail SMTP: reliable password reset and inquiry email delivery.
- One backup plugin (recommended: UpdraftPlus): rollback safety.
- One security plugin (recommended: Solid Security or Wordfence): login and hardening controls.

## Conditional
- Advanced Custom Fields: required only if editors need structured fields for blueprint/story/assets.
- Redirection: useful when managing many legacy URLs.

## Optional (choose one per category)
- SEO: Yoast SEO or Rank Math.
- Cache: LiteSpeed Cache (if supported) or WP Super Cache.
- Image optimization: ShortPixel or Smush.

## Avoid
- Multiple plugins in the same capability area.
- Checkout/cart enhancement plugins (catalog mode intentionally disables purchasing).
- Heavy page-builder plugins unless there is a hard content-editing requirement.
